# Baysn Internal Security Audit – September 9, 2025

## Overview
As part of our commitment to transparency and security, Baysn conducted an audit in our **2025 year**. The audit was designed to assess the security of our systems, including access to sensitive files, user data, and database protection.

## What Was Tested
The audit focused on the following areas:  

1. **.env File Access** – Checking if configuration files could be accessed.  
2. **Source Code Access** – Ensuring our codebase is secure.  
3. **User Data Access** – Assessing the security of stored user information.  
4. **Database Password Security** – Testing password protection measures.

## Key Findings
- **.env File:** Secure, no access found.  
- **Source Code:** No vulnerabilities discovered.  
- **Database Passwords:** Strongly protected.  
- **User Data:** Some historical user data was identified from early AI chat feature testing.  

The discovery of this data was made by our **Chief Information Security Officer (CISO), NetByte**, and was promptly addressed.

## Actions Taken
- The issue was reported internally and addressed immediately.  
- Measures have been implemented to prevent future occurrences.  
- We remain committed to safeguarding user data and maintaining transparency with our community.

## Our Commitment
Baysn takes security and privacy seriously. We conduct regular audits and continuously improve our systems to protect user information.
